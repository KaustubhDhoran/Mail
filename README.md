# Mail
A Java‑based console application that simulates core Gmail functionalities such as sending, receiving, and managing emails. Designed as a learning project to understand database management, query parsing, and CRUD operations in a real‑world context.
# Features
* Compose & Send Emails – Create new messages with subject, body, and recipients.

* Inbox Simulation – Store and view received emails.

* Delete Emails – Remove specific messages by ID or sender.

* Search Emails – Query by sender, subject, or keywords.

* Statistics – Count total emails, display oldest/newest, and track usage.

* Help Console – Built‑in guide for available commands.

# Tech Stack

Language: Java

Data Structures: LinkedList, HashMap for email storage

Environment: Console‑based application (runs in terminal/command prompt)
